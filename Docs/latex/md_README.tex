The \char`\"{}\+Human Tracker for Robots\char`\"{} project is a cutting-\/edge proposal presented to Acme Robotics aimed at addressing the challenge of \char`\"{}\+Human Detection and Tracking\char`\"{} within an indoor office environment using a specially designed robotic module equipped with a monocular video camera. The primary goal of this project is to develop an advanced robotic system capable of identifying and tracking humans within an office setting. This project leverages computer vision and machine learning. This is an efficient solution for various applications, such as enhancing security, improving workplace safety, and automating certain tasks within an indoor environment. Main features of the module include Human detection and Real-\/time Tracking. The Performance of the module is measured based on the detection accuracy and tracking precision.\hypertarget{md_README_autotoc_md3}{}\doxysubsubsection{Team}\label{md_README_autotoc_md3}
